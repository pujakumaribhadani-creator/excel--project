🛒 Ecommerce Website Logs Analysis Dashboard (Excel)
📌 Project Overview

This project presents a data analysis dashboard built using Microsoft Excel based on an Ecommerce Website Logs dataset.

The dataset contains detailed information about user activity, sales transactions, membership types, and product returns. The purpose of this project is to analyze customer behavior, sales performance, and return patterns using data cleaning, transformation, and visualization techniques in Excel.

📂 Dataset Information
Source: Ecommerce Website Logs Dataset
Original Dataset: 99,458 rows × 16 columns
Cleaned Dataset: 84,715 rows × 11 columns
Tool Used: Microsoft Excel
📑 Dataset Columns
Column Name	Description
Accessed Date	Date when the platform was accessed
Accessed From	Browser or App used
Age	User age
Age Category	Grouped age segment
Gender	User gender
Country	User location
Membership	Membership type
Sales	Purchase amount
Returned	Product return status
Returned Amount	Amount refunded
Payment Method	Mode of payment
