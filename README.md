# excel--project  Ecommerce Website Logs
# Project Overview
This project analyzes an Ecommerce Website Logs dataset using Microsoft Excel. 
The dataset contains information about user activity, sales transactions, membership types, and product returns on an ecommerce platform.

## Dataset Information
- Original Dataset: 99,458 rows and 16 columns
- Cleaned Dataset: 84,715 rows and 11 columns
- Tool Used: Microsoft Excel

Data cleaning and filtering were performed to remove unnecessary records and improve data quality. 
An additional column *"Age Category"* was created to analyze sales performance across different age groups.

## Columns in the Dataset
- Accessed Date
- Accessed From
- Age
- Age Category
- Gender
- Country
- Membership
- Sales
- Returned
- Returned Amount
- Payment Method

## Key Insights
- Sales distribution based on membership type
- Product return analysis
- Top 10 countries generating the highest sales
- Browser and app usage analysis
- Age-wise sales performance analysis

## Purpose of the Project
The purpose of this project is to analyze ecommerce user activity and sales patterns using Excel, and to gain insights into customer behavior, membership trends, and product returns.
